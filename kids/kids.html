Kids Gaming

Catch the Star Game

The Catch the Star game teaches basic game programming using keyboard control and scoring systems. Players move the character left and right to catch falling stars and increase their score in a fun and interactive way.

Snake Game

The Snake game helps children learn coding logic through movement, direction control, and collision detection. Players use arrow keys to control the snake, eat food, grow longer, and try to score as high as possible without hitting the wall.

Space Shooter

Space Shooter is an action game where the player controls a spaceship and shoots incoming enemies from the sky. The goal is to survive as long as possible while scoring points by destroying enemy ships.
